# PyPromise
JavaScript-alike promises in python

## How to set this up ##
Simply download the following files:
```
PyPromise.py
```

Add it either to your sys.path, or move it to where you want to use it.
Usage:
```py
import PyPromise

PyPromise.Promise(lambda: "Hello")
```
